Comics Reader

使用網頁技術精心打造的漫畫閱讀器

特色

  • 在 App 中搜尋、收藏漫畫
  • 介面簡約，無廣告干擾，沉浸的閱讀體驗
  • 跨平臺，支援 Windows、macOS、Linux
  • 漫畫更新通知，不錯過任何新漫畫
  • 漫畫備份下載功能
  • 剪取工具，蒐集有趣素材

為此專案貢獻

首先感謝 zeroshine 大大的 ComicsScroller，本專案一開始就是從 fork 出他的專案開始研究起的，後來為了練習 redux 架構以及嘗試 electron，慢慢的發展成現在的樣子。

身為一個開源的專案當然歡迎貢獻 :p，可以在本專案的 GitHub Issues 提出可以改進的之處，或是有趣的新功能建議。
