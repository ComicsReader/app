沒有找不到的漫畫，只有 DM5 沒有的漫畫*

簡而言之就是漫畫搜尋功能，由於是串接 DM5（一個第三方幫開 API 的概念），所以他沒有的漫畫本軟體就找不到。未來會不會支援其它站點，應該會吧

其它功能

容我再放一下首頁圖 😅，真的挺漂亮的（自己說）

以下是其它的特色以及其它有可能開發的功能，畫一塊大餅的意思。

  • 跨平臺，支援 Windows、macOS、Linux
  • 漫畫更新通知，不錯過任何新漫畫
  • 漫畫備份下載功能（規劃中）
  • 剪取工具，蒐集有趣素材圖（規劃中）

    • Contribution

    首先感謝 zeroshine 大大的 ComicsScroller，本專案一開始就是從 fork 出他的專案開始研究起的。
    後來為了練習 redux 架構以及嘗試 electron，慢慢的發展成現在的樣子。

    身為一個開源的專案當然歡迎貢獻啦，可以在本專案的 GitHub Issues 提出可以改進的之處，或是有趣的新功能建議。
    開發者就直接看 GitHub 的 README 辣 XDDD！
