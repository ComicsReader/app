# comics-dm5

An api wrapper for dm5.com.

## Usage

Install it through npm:

```sh
npm install comics-dm5 --save
```

```js
// es6 style
import * as dm5 from 'comics-dm5';

// or es5 style
const dm5 = require('comics-dm5');
```

## Development

```sh
npm install # install dependencies
npm run build # build
```
